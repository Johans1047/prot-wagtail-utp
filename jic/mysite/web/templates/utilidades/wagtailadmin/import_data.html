Instrucciones para importar

El archivo unificado debe contener estas columnas exactas:

ColumnaRequeridoDescripción
tituloSíTítulo de la investigación
añoSíAño del proyecto (ej. 2024)
universidadSíUniversidad participante
siglasNoSiglas de la Universidad (ej. UTP)
resumenNoDescripción del proyecto
categoriaNoCategoría del proyecto dentro de JIC
premioNo0=No ganador, 1=Primer lugar, 2=Segundo lugar, 3=Tercer lugar
asesorNoNombre exacto del Asesor/Consultor
emailNoCorreo electrónico del asesor
institucionNoInstitución a la que pertenece el asesor
activoNotrue/false o sí/no (Estado del asesor)

Notas de rendimiento:

  • Los nombres de columnas deben estar en minúsculas y sin espacios (para año puede usar 'ano' o 'año').
  • Con un solo archivo se cargarán y enlazarán los proyectos con sus respectivos asesores.
  • Habilitar la opción "Actualizar existentes" usará el nombre (Asesores) o título+año (Proyectos) para reemplazar datos antiguos.
  • Si no se habilita la deduplicación, omitirá automáticamente los registros que considere duplicados para proteger la base de datos de sobreescritura.
